# 🔐 Sistema de Controle de Acesso – Projeto de Testes QA

Projeto desenvolvido para simular o controle de acesso a uma sala de servidores, aplicando conceitos de testes automatizados com Jest.

## 📋 Regras de negócio

O acesso é permitido apenas se:

- Cargo for Técnico de Infraestrutura ou Administrador de Sistemas
- Horário entre 7h e 19h (inclusive)
- Funcionário não estiver na lista de bloqueio

## 🧪 Tipos de testes aplicados

- Testes positivos
- Testes negativos
- Testes de fronteira (Boundary Value Analysis)
- Validação de regras de negócio

## 🛠 Tecnologias utilizadas

- JavaScript
- Node.js
- Jest
